# Newton_ODsystem

功能概述：攻击端一键完成攻击演示，展示端通过更直接的方式展现黑客攻击威胁，可自主增加相应漏洞

### 牛顿应用场景：
- 企业安全部门成员向企业内其他部门成员、领导，展示黑客攻击的威胁，演示漏洞的危害，更好的促进企业安全发展
- 信息安全专业人员，向非相关专业小伙伴展示漏洞的危害，进行信息安全知识科普
